## Install

<Callout type="warning" emoji="⚠️️️">
  To use **Oh-my-fish**, you need install **Fish** shell fist.
</Callout>

As you can find on official repository, for install the **Oh-my-fish**, you only need to run the following command:

```shell
curl https://raw.githubusercontent.com/oh-my-fish/oh-my-fish/master/bin/install | fish
```

## Commands

The first command I use is the `help` command, so: `omf help` - this will list all the commands and options for **Oh-my-fish**.
The list of commands below are the commands I most use.

- **Install new package:** `omf install <package>` <small>(you can install multiple packages with only one command)</small>;
- **Search for packages:** `omf search <name-or-partial-name>`;
- **List local/installed packages:** `omf list`;
- **Remove/uninstall packages:** `omf remove <package>`;
- **List themes:** `omf theme`
- **Update Oh-my-fish:** `omf update`;

## Config

This is my personal configuration for **Oh-my-fish**, here you can find the alias and functions I created for my daily usage.

> **File:** `.config/fish/conf.d/omf.fish`

<Callout type="warning" emoji="⚠️️️">
If you pretend to use any of my alias/functions, you need to add that specific code after the 4 fist lines and before of the last line (`source ...`)
</Callout>

### Git Alias & Functions
```bash
alias g="git"
alias gst="git status -sb"
alias gp="git push"
alias gl="git pull"
alias gco="git checkout"
alias gb="git branch"
alias gaa="git add ."
alias ga="git add"
alias grh="git reset --hard"
alias gr="git reset"

function goMain
    git checkout main
    git pull
end

function goMaster
    git checkout master
    git pull
end

function goBranch
    git checkout "$argv"
    git pull
end

function commt
    git add .
    git commit -m "$argv"
    git push
end
```

### Docker Alias
