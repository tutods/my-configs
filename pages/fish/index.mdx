## Install

I'm only show the installation guide for **macOS** and **Linux**, but you can easier found the steps for other operating systems on the official
website.

<Tabs items={['macOS', <>Linux <small>(Debian distros.)</small></>]}>
    <Tab>
        If you are a **macOS** user, you probably have **[Homebrew](https://brew.sh/)** installed, so, only one command
        on your terminal and it's done!

        ```shell copy
        brew install fish
        ```
    </Tab>
    <Tab>
        The steps to install the **fish** shell are only 3, and you only need a terminal app.

        **1.** Install **fish** repository:

        ```shell copy
        sudo apt-add-repository ppa:fish-shell/release-3
        ```

        **2.** Update/upgrade repository:

        ```shell copy
        sudo apt-get update && sudo apt-get upgrade
        ```

        **3.** Install **fish** shell:

        ```shell copy
        sudo apt-get install fish
        ```

    </Tab>

</Tabs>

## Set as default shell

If you want to use **fish** as your default shell you need to run the following command:

```shell copy
echo /usr/local/bin/fish | sudo tee -a /etc/shells
chsh -s /usr/local/bin/fish
```

> **Remember:** `/usr/local/bin/fish` is the path of your **fish** installation. To get the path run `which fish`.

### Common Problems

<Tabs items={['Visual Studio Code', <>Hyper <small>(terminal)</small></>]}>
    <Tab>
        <Callout>
            If on your **Visual Studio Code** the shell don't change please follow the steps
            below.
        </Callout>

        1. Open your **Visual Studio Code**;
        2. Open the `settings.json` (not the settings UI);
        3. Add the code inside the **JSON** curly brackets.

        <Accordion title={"macOS"} className={'mt-2'}>
            <>
                ```json copy
                {
                    // ...
                    "terminal.integrated.defaultProfile.osx": "fish",
                }
                ```
            </>
        </Accordion>
        <Accordion title={"Linux"} className={'mt-2'}>
            <>
                ```json copy
                {
                    // ...
                    "terminal.integrated.defaultProfile.linux": "fish",
                }
                ```
            </>
        </Accordion>
    </Tab>
    <Tab>
        <Callout>
            If you are using **Hyper** terminal, and the shell don't change please follow
            the steps below.
        </Callout>

        1. Open in your **IDE** the file `.hyper.js` (is in your user root folder);
        - Normally, I do: `code ~/.hyper.js` to open in my **Visual Studio Code**.
        1. Search for `shell:` inside the `config:` object;
        1. Change for your **fish** path.
        - **Example:** `shell: '/usr/bin/fish',`;
        - **Remember:** `/usr/bin/fish` is the path for the **fish** installation (you can get it using `which fish`).

    </Tab>

</Tabs>
