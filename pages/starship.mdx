## How to install

Below you can find how to install the **Starship** on **macOS** and **Linux**. If you are a **Windows** user, please see the instructions on the
[official website][starship-url].

<Accordion title={'Linux & macOS'} className={'mt-2'}>
    If you are a **Linux** or a **macOS** user, you need to run the following command on your terminal:

    ```bash
    curl -sS https://starship.rs/install.sh | sh
    ```

</Accordion>

[starship-url]: https://starship.rs/
[nerdfonts-url]: https://www.nerdfonts.com/

## Setup shell to use **Starship**

<Accordion title={'Fish'} className={'mt-2'}>
	<div>
		<ol>
			<li>
				Open the <strong>Fish</strong>configuration file in your <strong>IDE</strong>;
				<ul className={'mt-0'}>
					<li>
						<strong>File location:</strong>
						<code>~/.config/fish/config.fish</code>
					</li>
				</ul>
			</li>
			<li>Add the code below on the end of the file.</li>
		</ol>
		<pre>
			<code className={'prism-code language-shell'}>starship init fish | source</code>
		</pre>
	</div>
</Accordion>
<Accordion title={'ZSH'} className={'mt-2'}>
	<div>
		<ol>
			<li>
				Open the <strong>ZSH</strong>configuration file in your <strong>IDE</strong>;
				<ul className={'mt-0'}>
					<li>
						<strong>File location:</strong>
						<code>~/.zshrc</code>
					</li>
				</ul>
			</li>
			<li>Add the code below on the end of the file.</li>
		</ol>
		```bash

            eval "$(starship init zsh)"
        ```
    </div>

</Accordion>
